# 🧠 fundamentos js - laboratorio 4 y 5

## 📌 descripción
este laboratorio tiene como objetivo aplicar los fundamentos de javascript en un entorno real, utilizando la consola del navegador (f12) sobre una página web existente.

para este caso, se trabajó sobre wikipedia, permitiendo interactuar directamente con el dom y modificar elementos en tiempo real.

---

## 🎯 objetivos

- comprender la sintaxis básica de javascript  
- aplicar conceptos fundamentales en una página real  
- manipular el dom desde la consola  
- utilizar herramientas del navegador para pruebas  
- evidenciar el comportamiento dinámico de javascript  

---

## 🌐 página utilizada

- https://www.wikipedia.org

se eligió esta página por su estructura clara y facilidad para manipulación mediante javascript.
